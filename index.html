Development Skills Inventory

Front End Projects

These two projects are front end projects I did to satisfy my own curiosity and get a better understanding of how the browser lays things out on the the screen.

CSS Is Awesome

Someone in my DigitalCrafts bootcamp circulated the CSS is Awesome meme while we were in the CSS portion of the class and indeed the message hit home for me. It inspired me to try and figure out if I could re-create the meme deliberately rather stumble upon it while studying CSS.

Bootstrap Mario

I wanted to get a better understanding of Bootstrap's grid model and one of the proposals was to generate an 8-bit Mario. I accomplished this by mapping a 2-dimensional array of "pixels" onto Bootrap's grid system.

Back End Projects

Twit-Ish

Twit-ish is a lightweight Twitter clone. This was used as my back end project submission for the DigitalCrafts bootcamp. It serves to combine a large number of technologies into a functional full-stack application including a Postgres database, authentication and authorization via bcrypt and JWT, and routing via Express, and fully online deploymenet via AWS EC2.
